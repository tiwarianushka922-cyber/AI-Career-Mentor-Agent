# AI Career Mentor Agent

## Problem
Students often do not know which career path to choose or what skills to learn.

## Solution
An AI-powered Career Mentor Agent that analyzes interests and skills, then recommends career paths and a personalized learning roadmap.

## Features
- Career recommendations
- Skill gap analysis
- Personalized roadmap
- Learning resource suggestions

## Technology
- Microsoft Foundry
- Foundry IQ
- GitHub
